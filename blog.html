The expiration date is the only significant distinction between localStorage and sessionStorage. Across browser sessions, localStorage maintains the store. Thus, the storage is kept even if the browser is closed and then opened again or the website is accessed in a new tab. There is no time limit. Whenever a page session expires, sessionStorage is deleted. As long as the tab or the browser is open, a page session is active. As a result, whenever you open a new tab or window, a new session and new storage are created.

The elements defined outside of any code blocks are all contained in the global scope. Statements clustered together and enclosed in curly braces make up a code block. Examples of a code block's structural elements include an if statement, loops, and functions. The scope of a global variable is universal. From any point in the code, a global variable is reachable. The items declared inside code blocks are included in the local scope, though. Local scope refers to a local variable. Only the place of declaration makes a local variable accessible.
